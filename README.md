# Installation

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```
## Cloning this repo
```cmd
> git clone https://github.com/naylachan/BOTZ
> cd BOTZ
```

## Install the package
```cmd
> npm i
```

## Edit config file


## Run the bot
```cmd
> npm start
```
